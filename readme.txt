A simple twitter widget that display most recent tweets from multiple accounts.

== Description ==
This wordpress Twitter plugin support multiple twitter accounts and displays the
tweets of the users in chronological order.

== Installation ==
Estimated Time: 5 minutes

1. Upload the 'widget-twitter-stream-wsyms' folder to the /wp-content/plugins/ folder on your site
2. In WP-Admin, you should see 'Multi Twitter Widget by wsyms' listed as an inactive plugin. Click the link to activate it. 
3. Once activated, go to Appearance > Widgets and Drop the Widget into your preferred sidebar
4. Once you've dropped in the widget, enter twitter handles (space separated) and click save.
